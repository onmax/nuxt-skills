# createReusableTemplate

Define and reuse template inside the component scope.

**Package:** `@vueuse/core`
**Category:** Component

## Usage

```ts
<template>
  <dialog v-if="showInDialog">
    <!-- something complex -->
  </dialog>
  <div v-else>
    <!-- something complex -->
  </div>
</template>
```
