# usePointerLock

Reactive pointer lock.

**Package:** `@vueuse/core`
**Category:** Sensors

## Usage

```ts
import { usePointerLock } from '@vueuse/core'

const {
  isSupported,
  lock,
  unlock,
  element,
  triggerElement
} = usePointerLock()
```
