# useWebWorker

Simple Web Workers registration and communication.

**Package:** `@vueuse/core`
**Category:** Browser

## Usage

```ts
import { useWebWorker } from '@vueuse/core'

const { data, post, terminate, worker } = useWebWorker('/path/to/worker.js')
```
