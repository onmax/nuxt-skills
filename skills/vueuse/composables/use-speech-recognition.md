# useSpeechRecognition

Reactive SpeechRecognition.

**Package:** `@vueuse/core`
**Category:** Sensors

## Usage

```ts
import { useSpeechRecognition } from '@vueuse/core'

const {
  isSupported,
  isListening,
  isFinal,
  result,
  start,
  stop,
} = useSpeechRecognition()
```
