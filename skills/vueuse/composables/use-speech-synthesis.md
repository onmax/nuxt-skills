# useSpeechSynthesis

Reactive SpeechSynthesis.

**Package:** `@vueuse/core`
**Category:** Sensors

## Usage

```ts
import { useSpeechSynthesis } from '@vueuse/core'

const {
  isSupported,
  isPlaying,
  status,
  voiceInfo,
  utterance,
  error,
  stop,
  toggle,
  speak,
} = useSpeechSynthesis()
```
