# useIntervalFn

Wrapper for with controls

**Package:** `@vueuse/shared`
**Category:** Animation

## Usage

```ts
import { useIntervalFn } from '@vueuse/core'

const { pause, resume, isActive } = useIntervalFn(() => {
  /* your function */
}, 1000)
```
