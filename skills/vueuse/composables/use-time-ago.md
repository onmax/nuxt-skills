# useTimeAgo

Reactive time ago. Automatically update the time ago string when the time changes.

**Package:** `@vueuse/core`
**Category:** Time

## Usage

```ts
import { useTimeAgo } from '@vueuse/core'

const timeAgo = useTimeAgo(new Date(2021, 0, 1))
```
