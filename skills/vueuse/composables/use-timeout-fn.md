# useTimeoutFn

Wrapper for with controls.

**Package:** `@vueuse/shared`
**Category:** Animation

## Usage

```ts
import { useTimeoutFn } from '@vueuse/core'

const { isPending, start, stop } = useTimeoutFn(() => {
  /* ... */
}, 3000)
```
