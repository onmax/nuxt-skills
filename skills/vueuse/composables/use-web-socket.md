# useWebSocket

Reactive WebSocket client.

**Package:** `@vueuse/core`
**Category:** Network

## Usage

```ts
import { useWebSocket } from '@vueuse/core'

const { status, data, send, open, close } = useWebSocket('ws://websocketurl')
```
