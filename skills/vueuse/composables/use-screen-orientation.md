# useScreenOrientation

Reactive Screen Orientation API. It provides web developers with information about the user's current screen orientation.

**Package:** `@vueuse/core`
**Category:** Browser

## Usage

```ts
import { useScreenOrientation } from '@vueuse/core'

const {
  isSupported,
  orientation,
  angle,
  lockOrientation,
  unlockOrientation,
} = useScreenOrientation()
```
