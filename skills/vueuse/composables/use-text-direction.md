# useTextDirection

Reactive dir of the element's text.

**Package:** `@vueuse/core`
**Category:** Browser

## Usage

```ts
import { useTextDirection } from '@vueuse/core'

const dir = useTextDirection() // Ref<'ltr' | 'rtl' | 'auto'>
```
